## DMIT 续费价格是怎么算的

先说最关键的一点：**DMIT 的循环折扣码，续费价格跟首购一样**。

这是 DMIT 在定价上做得比较厚道的地方。很多服务商惯用的套路是首年给你打七折、续费恢复原价，等你用习惯了再让你多掏钱。DMIT 不这样——只要你购买时用的是带 `RECURRING` 字样的优惠码，后续每个计费周期续费，都是按折扣价走，不会恢复原价。

举个例子，如果你当初买洛杉矶 EB 系列用了 `LAX-EB-LAUNCH-NON-MONTHLY-RECURRING-20OFF` 这个码，打了八折，那你每一次续费账单也是这个价格，不会在第二年突然跳回原价。

---

但有一个地方要注意。DMIT 官网的政策里写得很清楚：**平台保留随时调整套餐价格的权利，但当前计费周期内的价格不会变**。意思是说，如果你是按季度或者年付的，这段时间的价格锁死了。到期续费那一刻，价格以当时官网为准。大多数情况下续费价不会变，但如果 DMIT 调整了某个套餐的定价，续费时会以新价格 × 你的折扣比例来算。

所以建议：**季付和年付相对更省心**，不只是因为折扣力度大，也因为锁价周期更长。

## 老用户续费时常见的几个判断题

### 「续费还是新开一台？」

这两件事的适用场景不一样。

**直接续费**适合：当前套餐配置够用、线路表现稳定、没有更好的促销价可以切换。直接续费最简单，也不用迁数据。

**考虑新开一台**适合：现有套餐的机房或线路不再满足需求（比如你从主要用洛杉矶改成主要跑国内流量，香港延迟对你更合适）；或者发现有一个新套餐配置更好、叠加优惠码后价格比你当前续费价更低。

需要说一下的是：如果新开套餐同时用了老用户码，实际成本未必比续费贵，有时候反而更低。

### 「当初没用优惠码，续费时可以补吗？」

续费订单不支持单独补码。优惠码必须在首次购买时输入，才能在后续周期自动应用。

如果当初裸价购买，续费会按原价走。想拿折扣价，就只能新开一个订单。

不夸张地说，这个细节很多人是在续费的时候才意识到的。所以如果你手里有正在续费的套餐，但当初没用码，后面如果有机会新开，记得先把可用的优惠码在购物车里验证一遍。

### 「套餐超量限速之后要不要升级？」

不用立刻升级。DMIT 的超量政策是**限速不断网**——月流量用完之后，带宽会降速到一个较低的档位（不同套餐不同，通常在 50Mbps–1Gbps 之间），不会直接切断服务。

如果你是轻度用户，偶尔超量，限速的那几天一般也够用。如果经常触发限速，说明当前套餐流量上限确实不够，这时候考虑升级或者换大流量套餐才有意义。

### 「IP 被墙怎么办，续费时会不会换 IP？」

续费不会自动换 IP。IP 是跟你的套餐绑定的。

DMIT 官方政策是：每 15 天可以免费申请一次 IP 更换（被 GFW 封锁的情况下），超出频率后每次收费 $5。有些机房套餐支持购买 Guarantee+ 附加选项，如果分配到的 IP 在敏感地区不可用，可以发工单免费重新分配。这个附加项是一次性付费，月付用户如果担心 IP 可用性，可以考虑加上。

## 钱的问题说直一点

续费价格这件事最容易被忽视的一个细节：**付费周期越长，折扣通常越大**，而且锁价时间也越长。

## 常见问题

**Q：续费时优惠码需要手动填吗？**

不需要。购买时用过的循环优惠码会自动绑定到这个订单，后续每次续费账单都按折扣价生成。不用你手动操作。

**Q：我的套餐到期前多久可以续费？**

DMIT 支持提前续费，到期前续费款项会叠加在当前到期时间之后。建议在到期前 1–2 周处理，避免因余额不足或支付问题导致服务中断。

**Q：老用户码 `2025-EXISTING-CUSTOMER-10OFF` 可以用在续费订单上吗？**

这个码针对的是**新购订单**，不是对已有订单的续费账单。如果你要用这个码，需要新开一台，而不是在已有套餐的续费界面使用。

**Q：DMIT 换套餐要删机重新开吗，数据怎么处理？**

套餐升级可以通过工单联系客服处理，不一定需要删机。但跨机房或跨产品线的切换通常需要新开，数据需要自己迁移。具体操作建议提工单确认，客服响应一般在半小时以内。

**Q：圣诞节那批特惠套餐还在卖吗？**

DMIT 的特惠套餐通常有数量限制，卖完就下架，不定期补货。建议直接去官网定价页看当前有哪些套餐在售，库存状态实时更新。
